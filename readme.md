# htb-cli

> My awesome CLI for HackTheBox.

## Features

- [x] Login with API Token
- [x] Machine list
- [x] Machine info
- [x] Machine list (Upcoming)
- [x] Machine Flag submit
- [x] Profile
- [x] Whoami
- [x] Arena info
- [x] Arena start
- [x] Arena submit
- [x] VPN Info

## Install

```bash
npm install --global https://github.com/BRAVO68WEB/htb-cli
```

### Useage

```bash
╰─$ htb-cli --help
Usage: htb-cli [options] [command]

Unofficial Hack The Box CLI client

Options:
  -v, --version   Show version number
  -h, --help      Show help

Commands:
  arena
  auth [options]
  machines
  profile
  re-auth
  whoami
  help [command]  Show help for command
```
